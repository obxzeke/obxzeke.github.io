Interface uses a Basys 3 board to simulate an Arithmetic Logic Unit using VHDL. Two 4-digit binary inputs are used as well as a 3-digit binary input to select the operation and then display the value to the operation using four 7-segment displays on the board.

Two-Dice Pig

C++ collaborative team project to recreate a popular dice game. The program used functions to roll dice for a player and to calculate the score for a player of that given roll.
